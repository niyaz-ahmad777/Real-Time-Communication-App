# Real-Time Communication App

This repository contains a basic real-time communication application with
separate `server` and `client` directories. The server uses Node.js, Express,
and Socket.IO for handling WebSocket connections, while the client is built
using React and Vite. The code is organized to support authentication,
WebRTC, and real-time messaging.

## Structure

```
rtc-app/
│  README.md
│
├─ server/  (Express + Socket.IO backend)
└─ client/  (React + Vite frontend)
```

Refer to the individual folders for setup instructions.
